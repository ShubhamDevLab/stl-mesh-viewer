# External Libraries

This folder contains third party dependencies required to build the STL Viewer.

## 📂 Structure

```
include/   → Header files
lib/       → Compiled libraries
```

## 📦 Required Libraries

Place the following:

### Headers (`include/`)

* glad
* GLFW
* glm
* stb

### Libraries (`lib/`)

* glfw3.lib

---

## ⚠️ Note

This folder is ignored in Git (`.gitignore`) to keep the repository clean.
Download dependencies manually before building the project.
